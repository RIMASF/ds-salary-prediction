# DS Salary Prediction

Predicting data science salaries using a Random Forest model trained on the `ds_salaries` dataset.

## Libraries

- pandas, numpy
- matplotlib
- scikit-learn

## Notebook Structure

| # | Section |
|---|---------|
| 1 | Import Libraries |
| 2 | Read Data |
| 3 | Data Cleaning |
| 4 | Feature Engineering |
| 5 | Remove Outliers (IQR) |
| 6 | X and y Split |
| 7 | Encoding |
| 8 | Train/Test Split |
| 9 | RandomForest Model |
| 10 | Prediction |
| 11 | Evaluation + Cross-Validation |
| 12 | Feature Importance |
| 13 | Analysis Questions |

## Analysis Questions

- Does experience level affect salary?
- Do remote jobs have higher salaries?
- Does company size affect salary?

## How to Run

```bash
pip install -r requirements.txt
jupyter notebook project.ipynb
```
